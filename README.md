## REPRO STEPS

1. go to `packages` or `apps` folder

2. create a Vite project
```bash
pnpm create vite
```

3. go to the Vite project

4. setup Storybook and you'll see the error.
```bash
pnpm create storybook@latest
```

5. check your pnpm registry to see if there're some errors (optional)
```bash
pnpm config get registry
```
